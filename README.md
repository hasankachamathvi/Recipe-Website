**Recipe World** is a modern, interactive web application that allows users to explore, view, and share delicious recipes. The website features a visually appealing homepage, featured recipe cards, and detailed recipe pop-ups for an immersive cooking experience.

---

## **Features**

- Modern homepage with hero section
- Featured recipe cards with images and descriptions
- View Recipe popup/modal to show full recipe details
  - Recipe name
  - Category (Dessert, Main Course, Drinks)
  - Ingredients
  - Steps
  - Image
- Responsive design for desktop and mobile
- Easy navigation between Home, Recipes, and Add Recipe pages

---

## **Technologies Used**

- HTML5
- CSS3 (with modern styling and hover effects)
- JavaScript (for modal popup functionality)
- Optional: Node.js / JSON backend for storing recipes
